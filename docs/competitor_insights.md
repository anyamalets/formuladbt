# Конкурентный анализ — выжимка для formuladbt.ru

Проанализировано 6 русскоязычных ДБТ-команд + 8 международных ДБТ-центров и mental-health сайтов. Полные отчёты двух агентов — в conversation transcript.

---

## Приоритизированные идеи (Quick wins сверху)

### 🟢 Очень высокий ROI + лёгкая реализация

**1. Блок «Когда ДБТ — не первый выбор» с альтернативами**
Источник: rusdbtteam.com/no_dbt
Зачем: для ПРЛ-аудитории, чувствительной к границам и страху «впихнут», это мощный сигнал доверия. Уникальная честность — никто из конкурентов кроме одной команды это не делает.
Реализация: 1 текстовый блок (3-4 пункта: «не возьмём, если активная анорексия / активный психоз / нет 6+ мес готовности → направим к коллеге»)

**2. Прозрачные цифры команды в hero/about**
Источник: Stanford, Yale
Сейчас: «Команда психологов с подготовкой по ДБТ» — расплывчато
Реализация: «4 терапевта, суммарно X лет в ДБТ, еженедельная супервизия» (без регалий-стены)

**3. Блок «Команда как метод» в секции About-DBT**
Источник: гибрид dbt-team.ru и dbteam.pro + DBT Vancouver
Цитата для усиления: «ДБТ не работает в одиночку — команда это часть метода»
Реализация: добавить в существующий «Что такое ДБТ» 1-2 абзаца про team consultation и зачем это клиенту

**4. Описать 4 модуля навыков через глаголы-действия**
Источник: DBT Vancouver
Сейчас: «осознанность, эмоциональная регуляция, межличностная эффективность, стрессоустойчивость» (термины)
Лучше: «управлять вниманием, справляться с эмоциями, эффективно вести себя в отношениях, переносить эмоциональный дистресс» — клиентский язык

**5. Сместить блок «После заявки» — поставить ДО формы**
Источник: 3East, mental-health design patterns
Зачем: тревожный клиент читает страницу сверху вниз, ему важно понимать «что будет дальше» ДО того как нажмёт submit. Сейчас «После заявки» идёт после формы — теряем доверие на момент решения.

### 🟡 Средний эффект + средние усилия

**6. Бесплатное 20-минутное ознакомительное собеседование**
Источник: dbtunity.com + Two Chairs «matching appointment»
Зачем: снимает порог «подойдёт ли мне», особенно для клиенток с 5 неудачными терапевтами
Реализация: текст в форме «первая встреча с координатором — бесплатно, 20 мин, не сессия» + слот через Calendly

**7. Двухшаговая воронка: форма → анкета подбора → чат в Telegram**
Источник: dbteam.pro
Зачем: лучше мэтчинг, меньше «не подошёл специалист»
Реализация: вторая короткая анкета на странице /thanks (запрос, диагноз/симптомы, опыт терапии, ЧП), потом ты вручную распределяешь

**8. Отдельная страница каждого терапевта (`/team/[slug]`)**
Источник: dbtufa.com
Зачем: глубже доверие + SEO по именам коллег
Реализация: 4 страницы по шаблону с расширенным био, образованием, супервизиями, специализацией, видео-знакомством (если есть)

### 🔴 Высокий импакт, высокие усилия (на будущее)

**9. Бесплатная мини-библиотека навыков ДБТ (3-5 аудио)**
Источник: impulse-dbt.ru/free-skills
Что: «Камешек в озере», «Дыхание мудрого ума», «STOP»
Зачем: лид-магнит + знакомство с голосом команды до заявки
Реализация: запись озвучек + блок-плеер + страница /skills

**10. Лендинг «Для близких» (родственников ПРЛ-клиента)**
Источник: dbtunity.com/fc
Что: чеклист «как поддерживать близкого с эмоциональной нестабильностью» + 2-3 навыка валидации
Зачем: расширение воронки на тех, кто гуглит «как помочь жене/дочери с ПРЛ»
Реализация: отдельная страница + PDF

## Анти-паттерны — то что увидели у конкурентов и НЕ повторяем

- «Поможем обрести себя / вернуть радость жизни» (dbteam.pro) — обещания результата
- «Мы рядом / вместе с вами» — встречается у нескольких, нам нельзя по PRL-правилам
- Размытие ДБТ в общем КПТ-зонтике (dbteam.pro) — теряется специализация
- Two Chairs стиль «Stephanie любит бегать с голдендудлем» в био — для ПРЛ/РПП = парасоциальность и размытие границ
- CMC стиль «kindness is the glue» — приторно, стоп-лист
- Stanford/Yale перечни 10+ диагнозов — для команды из 4 человек = переобещание
- Длительность «1 год» подана как факт без объяснения (dbtunity.com) — пугает
